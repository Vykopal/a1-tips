transcludeFn in link function

Let us look here:

There is a way of how to control transclusion from JS inside the link function of a directive. But - it is tightly coupled with $compileProvider API which is not supported in Angular2. So, if you wanna migrate your directives (application) to Angular2 later, just forgot about it. The Angular2 compiler and DOM manipulation works in a completely different way. And - everything you can do inside the link function using transcludeFn is automatically done by ngTransclude directive.

My advice is: use ngTransclude directives when apropriate, but do not bother with the low level $compile API and things like transcludeFn in link function - it brings you no benefit, and will be discarded when upgrading your app to Angular2.
