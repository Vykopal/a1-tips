Pipes and filter performance

open console

View version "pipe.companies | findKey: 'FO' : 'view'"
Company: {{pipe.companies | findKey: 'FO' : 'view'}}

Model version "findKeyFilter(this.companies, 'FO', 'controller')"
Company: {{pipe.company}}

Model version "$filter('findKey')(this.companies, 'FO', 'controller')"
Company: {{pipe.company}}

If the pipe filters object or array, it gets executed on every $digest loop! Only for primitive values it doesn't.

If you need to filter large arrays or objects, do so in controllers, it gets executed once!
